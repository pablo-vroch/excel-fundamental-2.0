### Diccionario de Propiedades Termodinámicas (Data Bank)

Esta tabla contiene todas las propiedades que pueden extraerse de la base de datos. Para llamar a una propiedad en una función, se puede utilizar cualquiera de los términos listados en la columna **Formas de referenciar**.
Es importante considerar que las formas de referenciar son flexibles, son indiferentes a espacios, mayusculas, acentos y guiones.

| Propiedad | Símbolo | Formas de referenciar |
| :--- | :---: | :--- |
| Fórmula química del compuesto | - | Fórmula, Formula, formula |
| Nombre principal del compuesto | - | Nombre, Name, Nombre IUPAC, nombre |
| Registro único CAS | - | CAS, Registro CAS, cas |
| Masa molar (Peso molecular) | $M_w$ | Peso Molecular, Masa Molar, Molecular Weight, Wt, wt |
| Temperatura de fusión normal | $T_f$ | Temperatura de Congelación, Tf, Freezing Temperature, tfp |
| Temperatura de ebullición normal | $T_b$ | Temperatura de Ebullición, Tb, Boiling Temperature, tbp |
| Temperatura en el punto crítico | $T_c$ | Temperatura Crítica, Tc, Critical Temperature, tc |
| Presión en el punto crítico | $P_c$ | Presión Crítica, Pc, Critical Pressure, pc |
| Volumen en el punto crítico | $V_c$ | Volumen Crítico, Vc, Critical Volume, vc |
| Factor de compresibilidad crítico | $Z_c$ | Compresibilidad Crítica, Zc, Z Crítica, zc |
| Factor acéntrico de Pitzer | $\omega$ | Factor Acéntrico, Omega, w, Acentric Factor |
| Entalpía estándar de formación | $\Delta H_f^\circ$ | Entalpía de formación, Enthalpy of Formation, delhform |
| Energía de Gibbs de formación | $\Delta G_f^\circ$ | Energía de Gibbs, Gibbs of Formation, delgform |
| Entalpía de vaporización (Calor latente) | $\Delta H_{vap}$ | Entalpía de Vaporización, Calor Latente, Hvap, delhb |
| Entalpía de fusión | $\Delta H_{fus}$ | Entalpía de Fusión, Enthalpy of Fusion, delhf |
| Volumen molar del líquido saturado | $V_{liq}$ | Volumen Líquido, Vliq, Liquid Volume, vliq |
| Temperatura de referencia para $V_{liq}$ | $T_{ref,L}$ | Temperatura Líquido, Liquid Temperature, tliq |
| Momento dipolar | $\mu$ | Dipolo, Dipole, dipole |
| Temperatura mínima para el modelo de $C_p$ | $T_{min,Cp}$ | Cp T-Min, Cp Temperatura Mínima, cptmin |
| Temperatura máxima para el modelo de $C_p$ | $T_{max,Cp}$ | Cp T-Max, Cp Temperatura Máxima, cptmax |
| Capacidad calorífica (Gas Ideal) | $C_p^{ig}$ | Cp Gas Ideal, Cp ig, cpgasideal |
| Capacidad calorífica (Líquido) | $C_p^{liq}$ | Cp Líquido, Cp liq, cpliquido |
| Constantes de Cp polinomial (Gas Ideal) | $C_p^{ig}(T)$ | Cp |
| Constantes de Antoine| ${A, B, C}$ | Antoine, Ant, ant |
| Temperatura mínima (Antoine) | $T_{min,Ant}$ | Ant-tmin |
| Temperatura máxima (Antoine) | $T_{max,Ant}$ | Ant-tmax |
| Presión de vapor mínima (Antoine) | $P_{min,Ant}$ | Ant-pvmin |
| Presión de vapor máxima (Antoine) | $P_{max,Ant}$ | Ant-pvmax |
